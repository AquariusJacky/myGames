# MyGames
This is going to be a combination of several minigames.

## Environment
Ubuntu 22.04.1 LTS
g++ 11.4.0

## Installation
```bash
sudo apt update
sudo apt-get install qt5-default // sudo apt-get install qtbase5-dev qtchooser qt5-qmake qtbase5-dev-tools
```

## Tic Tac Toe
Implementation of the minimax algorithm.
